# Dados do Projeto – Otimização de Concreto (PSO)

Este diretório contém os arquivos oficiais fornecidos pelo projeto GENIAL (UFPA):

- `concrete_data.csv`: Base de dados com 1030 amostras contendo componentes do concreto e sua resistência.
- `concrete_rf.pkl`: Modelo Random Forest pré-treinado utilizado para prever a resistência à compressão.
